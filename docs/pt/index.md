**Diretrizes do IGS para Estações de Referência Operacionais Contínuas (CORS)**

*Versão 1.0.0-pt*

[Adira à Rede IGS](https://igs.org/network-resources/#propose-new-site){ .md-button .md-button--primary }

A rede do IGS é um conjunto de Estações de Referência Operacionais Contínuas (CORS) mantidas e operadas por uma variedade de organizações diferentes que juntam os seus recursos sob a égide do IGS para o bem comum. Regras estritas são incompatíveis com a natureza voluntária do IGS. Contudo, as estações participantes devem aderir a um conjunto de normas e convenções acordadas, que garantem o fornecimento de dados consistentes e adequados à finalidade. De particular importância para os produtos do IGS é o funcionamento estável e a longo prazo da rede de rastreio. Portanto, quaisquer alterações à configuração da estação ou ao ambiente circundante devem ser cuidadosamente planeadas para minimizar descontinuidades na série temporal de posições da estação. Deve ser dada especial atenção às estações de referência designadas que contribuem para a realização do Sistema de Referência Terrestre Internacional[^1] (ITRF).
Este documento lista as convenções mínimas que todas as CORS do IGS devem seguir, bem como características adicionais desejáveis que aumentam o valor de uma estação para o IGS. Destina‑se a ser uma referência útil para proprietários e operadores de estações na planificação e operação das CORS. Sugestões para adições ou alterações a estas Diretrizes são bem‑vindas em [cb@igs.org](mailto:cb@igs.org).

## Procedimento para se Tornar uma Estação IGS

O processo que os proprietários ou operadores de estações devem seguir para propor uma nova estação IGS está descrito no documento “[Procedure for Becoming an IGS Station](https://files.igs.org/pub/resource/guidelines/Procedure_for_Becoming_an_IGS_Station.pdf)”. Ao planear a proposta de uma nova estação, o operador deve rever cuidadosamente as diretrizes contidas neste documento. A agência responsável deve ter a expectativa de que a estação irá operar perpetuamente, preferencialmente como parte de uma rede de referência nacional ou regional. As estações localizadas no âmbito de uma rede de referência nacional ou regional devem primeiro coordenar‑se com o coordenador dessa rede[^2]. Uma vez que a maioria das redes de referência regionais ou nacionais se alinham com os padrões do IGS, a aceitação prévia dentro dessas redes é considerada favorável ao IGS. A SPC (Comissão de Propostas de Estações), uma sub-comissão da Comissão de Infraestruturas do IGS, decidirá se a estação proposta deve ser aceite na rede IGS com base na localização, instrumentação, características operacionais e se é relevante para qualquer Projecto Piloto ou Grupo de Trabalho do IGS.
O Coordenador da Rede IGS coordena com o operador da estação o processo de candidatura revendo todas as informações compiladas relacionadas com a precisão, completude e conformidade com estas Diretrizes.

## Dispensas para Estações não Conformes

Se uma estação não cumprir todas estas Diretrizes, mas ainda assim for valiosa para o IGS, pode ser concedida uma dispensa para incluir a estação na rede do IGS, caso seja acordado pelo SPC.
